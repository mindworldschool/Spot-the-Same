# ⚠️ Важно: Добавьте изображения символов!

Папка `assets/symbols/` должна содержать изображения символов из вашего архива.

## Что нужно сделать:

1. **Распакуйте архив `images.zip`** с символами
2. **Скопируйте все WebP файлы** в эту папку (`assets/symbols/`)
3. Убедитесь что названия файлов совпадают с `symbols.json`

## Структура должна быть:

```
assets/
  symbols/
    abacus.webp
    backpack.webp
    ball.webp
    bell.webp
    bird.webp
    book.webp
    brain.webp
    ... (всего 58 файлов)
```

## Проверка:

После добавления файлов откройте `index.html` в браузере.
Если символы не загружаются, проверьте:
- ✅ Пути к файлам правильные
- ✅ Названия файлов совпадают с `data/symbols.json`
- ✅ Файлы в формате WebP (или PNG как fallback)

---

**Примечание:** Символы не включены в репозиторий, чтобы избежать проблем с авторскими правами.
